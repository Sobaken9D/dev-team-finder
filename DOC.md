**Dev Team Finder** — это платформа-связка для IT-сообщества. Мы помогаем
джуниорам и разработчикам без коммерческого опыта найти свою первую команду или
ментора, а опытным командам — найти мотивированных помощников для своих
проектов.

### Функционал приложения

1. **Поиск проекта**
    - Система заявок (возможность откликнуться на вакансию в проекте).
    - Фильтрация проектов
        * по тематике
        * по стеку
        * по уровню
        * по занятости
        * по часовому поясу
        * по требуемой роли
    - Карточка проекта (превью с основными метками)
      * название
      * стек
      * уровень
      * картинка
      * краткое описание
    - Быстрый отклик.
    - Расширенный отклик (с письмом, анкетой, интервью и тд.).
2. **Поиск команды**
   - Создание вакансии
     * роль
     * количество мест
     * стек для этой роли
     * описание задач
     * уровень кандидата
   - Управление откликами
     * входящие заявки (список кандидатов)
     * возможность принять/отклонить заявку
     * возможность начать личный чат
     * просмотр профиля кандидата
3. **Создание собственного проекта**
    - Информация о проекте
      * название
      * описание
      * информация для фильтрации (тематика, ...)
      * картинка (логотип или другое)
      * участники
    - Настройка команды
      * участники
      * роли участников
      * задачи участников
    - Статус проекта (стадия или активность).
4. **AI ассистент**
    - Для поиска (рекомендации под твой профиль).
    - Подсказки.
    - Аналитика.
    - Генерация ивентов/соревнований.
5. **Профиль**
    - Аватар.
    - Никнейм.
    - Действующие проекты.
    - Возможность прокачки игрового аватара за счет ивентов, решения задач и тд. 
    - Информация о себе
      * уровень
      * опыт
      * языки
      * фреймворки
      * имя
      * фамилия
      * страна
      * образование
      * контакты (telegram, git-hub)
      * о себе
      * пет проекты
      * уровень английского
6. **Чаты/обсуждения внутри платформы**
    - Личные сообщения.
    - Групповые чаты (чат проекта и тд.).
    - Отправка текстов, изображений, файлов.
    - Опросы, голосования.
    - Реакции.
7. **Система ивентов/соревнований**
    - Хакатоны.
    - Митапы.
    - Вебинары.
    - Соревнования.
    - Код-ревью.
    - Функционал ивента
      * страница ивента с информацией
      * регистрация на ивент
      * чат ивента
      * расписание
      * формирование команд
      * создание ивента
      * баллы за участие
      * топ участников
    - Рейтинг и достижения.
8. **Раздел полезных материалов**
    - Гайды.
    - Шаблоны (сопроводительные письма, CV)
    - Подборки ресурсов.
    - Статьи.
    - Функция загрузки материалов.
    - Лайки, комментарии.
